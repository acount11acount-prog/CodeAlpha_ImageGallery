# Image Gallery Web App

## Overview

This project is a **Simple Image Gallery** developed using **HTML, CSS, and JavaScript**.
It displays a collection of images in a visually organized layout and allows users to view images interactively.

The goal of the project is to practice **frontend development concepts such as responsive layouts, DOM manipulation, and UI design**.

## Features

* Grid-based image gallery layout
* Responsive design for different screen sizes
* Clickable images for better viewing experience
* Smooth UI and organized image display

## Technologies Used

* **HTML5** – Structure of the gallery page
* **CSS3** – Styling and layout design
* **JavaScript** – Interactive behavior and image actions

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.

No installation or external libraries are required.

## Learning Objectives

This project helps in understanding:

* Responsive grid layouts
* Image display optimization
* JavaScript event handling
* Basic UI/UX design principles

## Limitations

* Images are static and locally stored.
* No dynamic image upload feature.
* No backend integration.
